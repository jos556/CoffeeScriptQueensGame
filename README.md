> 完成遊戲時的勝利畫面

## 🎯 遊戲規則

1. 每行只能放置一個皇后 (👑)
2. 每列只能放置一個皇后
3. 每個顏色區域只能放置一個皇后
4. 皇后之間不能相鄰（包括對角線）
5. 點擊一次放置 X，再次點擊放置皇后 (👑)
